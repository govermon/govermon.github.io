What this page is

A static project page published from the public govermon.github.io repository. It can point at the current VM IP without asking the public project URL to change every time the infrastructure changes.

Current hosting model

The app is served through Caddy on a single VM, exposed over a reserved static IP, and tracked by a small sanitized status file exported from the private source repo.

What is missing without DNS

No memorable hostname, no automatic public TLS certificate for the app URL, and less user-friendly sharing. The landing page reduces link churn, but it does not replace DNS.

Publishing model

The private source repo exports this page, its project metadata, and the deployment status payload into the public govermon.github.io repository through an automated GitHub App-backed sync workflow.
